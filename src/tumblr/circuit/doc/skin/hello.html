  • Go Circuit Distributed computing extension of the Go language and runtime

    DCCP is a standardized hand-shake and congestion-control protocol at the IP-level, also to be found in the Linux kernel.

  • Vitamix compiler Time virtualization for Go

    Vitamix is a source-to-source (Go-to-Go) transformer. It rewrites Go code so it executes in an environment with a customizable notion of time, hence the term “time virtualization”. I use it in developing congestion control algorithms, where virtualizing the original code allows me to run simulations, that take hours of real time, instantaneously.
