Go Circuit

Introduction

In implementing a distributed application, one should be able to express the intra-host as well as inter-host program logic within a common, “uniform” linguistic environment — much like a Go program is a common and sufficient language for expressing the intra-goroutine and inter-goroutine logic of a complete application.

The Go Circuit attempts to extend the reach of the Go Language to distributed computation. The goal of this project is to build a programming environment that facilitates both the development and the life-long maintenance of cloud applications and even interdependent multi-application systems.
